# AI Creator Hub (React Version)

React重构版AI Creator Hub，功能与原HTML版本一致，但交互性更强。

## 启动

```bash
cd ai-creator-hub-react
npm install
npm run dev
```

访问: http://localhost:3002

## 构建生产版本

```bash
npm run build
```

输出到 `dist/` 目录，可直接部署到服务器。

## 功能

- 6个AI工具页面
- MetaMask钱包连接
- React Router SPA导航
- 响应式设计
- 深色主题 + 渐变配色
